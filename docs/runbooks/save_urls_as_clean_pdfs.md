# Runbook: save_urls_as_clean_pdfs.py

## Description
Converts a list of URLs into simplified, reader-mode PDFs. It archives the substantive content of pages without ads, navigation, or chrome.

## Prerequisites
- **Python 3.x:** Must be installed.
- **Dependencies:** `trafilatura` and `weasyprint` (installed via `poetry install` or `pip`).
- **Input File:** `local/brave/urls.txt` (generated by `extract_urls_from_brave.py`).

## Usage
1. Ensure `local/brave/urls.txt` exists.
2. Run the script:
   ```bash
   python tools/save_urls_as_clean_pdfs.py
   ```
3. PDFs will be saved to `local/brave/pdfs/`.

## Logic
- Uses `trafilatura` to extract clean article text and metadata.
- Uses `weasyprint` to render the extracted HTML/content into a PDF.
